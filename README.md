# WOSSFOSS Vapoursynth

## How do I install these formulae?

`brew install wossfoss/vapoursynth/<formula>`

Or `brew tap wossfoss/vapoursynth` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "wossfoss/vapoursynth"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
